DETALLES DE CONEXIÓN
═══════════════════════════════════════════════════════════════════

┌─ BOTONES (x4) ────────────────────────────────────────────────┐
│                                                                │
│   ESP32 GPIO 12/13/14/15                                       │
│        │                                                       │
│        └─── [Botón] ─── GND                                   │
│                                                                │
│   Nota: Se usa INPUT_PULLUP interno, no requiere              │
│         resistencia pull-up externa                           │
│                                                                │
│   Estado normal: HIGH (3.3V)                                  │
│   Presionado: LOW (0V)                                        │
└────────────────────────────────────────────────────────────────┘

┌─ LEDs (x4) ───────────────────────────────────────────────────┐
│                                                                │
│   ESP32 GPIO 16/17/18/19                                       │
│        │                                                       │
│        └─── [Resistencia 220Ω] ─── [LED Ánodo (+)]           │
│                                         │                      │
│                                    [LED Cátodo (-)]           │
│                                         │                      │
│                                        GND                     │
│                                                                │
│   Colores sugeridos:                                          │
│   - LED 1: Rojo    (Jugador 1)                               │
│   - LED 2: Azul    (Jugador 2)                               │
│   - LED 3: Verde   (Jugador 3)                               │
│   - LED 4: Amarillo (Jugador 4)                              │
│                                                                │
│   Nota: La pata larga del LED es el ánodo (+)                 │
└────────────────────────────────────────────────────────────────┘

┌─ BUZZER ──────────────────────────────────────────────────────┐
│                                                                │
│   ESP32 GPIO 25                                                │
│        │                                                       │
│        └─── [Buzzer +] (Terminal positivo)                    │
│                                                                │
│        GND ─── [Buzzer -] (Terminal negativo)                 │
│                                                                │
│   Tipos de buzzer:                                            │
│   - Activo: Hace sonido al recibir voltaje                    │
│   - Pasivo: Requiere señal PWM para frecuencias               │
│                                                                │
│   El código usa buzzer pasivo (permite melodías)              │
└────────────────────────────────────────────────────────────────┘

┌─ SERVOS (Opcional) ───────────────────────────────────────────┐
│                                                                │
│   Servo 1:                      Servo 2:                      │
│   ┌────────┐                    ┌────────┐                    │
│   │ Marrón │ ─── GND            │ Marrón │ ─── GND           │
│   │ Rojo   │ ─── 5V             │ Rojo   │ ─── 5V            │
│   │ Naranja│ ─── GPIO 26        │ Naranja│ ─── GPIO 27       │
│   └────────┘                    └────────┘                    │
│                                                                │
│   ⚠️  IMPORTANTE:                                              │
│   Los servos consumen mucha corriente (~500mA bajo carga)     │
│   NO conectar directamente al pin 5V del ESP32                │
│   Usar fuente externa 5V 2A+ con GND común                    │
│                                                                │
│   Conexión con fuente externa:                                │
│   ESP32 GND ─┬─ Servo GND                                     │
│              └─ Fuente GND                                    │
│                                                                │
│   Fuente 5V ──── Servo VCC                                    │
│   ESP32 GPIO 26/27 ──── Servo Signal                          │
└────────────────────────────────────────────────────────────────┘

NOTAS DE SEGURIDAD
═══════════════════════════════════════════════════════════════════

⚠️  ADVERTENCIAS:
  • No conectar 5V directo a pines GPIO (max 3.3V)
  • No sobrepasar 40mA por pin GPIO
  • Usar resistencias con LEDs (protección)
  • GND común en todas las conexiones
  • Desconectar alimentación antes de modificar circuito
  • Verificar polaridad de componentes

✅ BUENAS PRÁCTICAS:
  • Usar cables de colores (rojo=VCC, negro=GND)
  • Etiquetar conexiones
  • Probar componentes individuales primero
  • Medir voltajes con multímetro
  • Documentar cambios
